# Excel Vendor Spend Analysis

## Overview
This project analyzes vendor invoice and payment data using Microsoft Excel.
It covers spend categorization, payment status tracking, and monthly trend
analysis built to mirror real FP&A reporting workflows.

## Business Questions Answered
- What is total spend and outstanding balance by department?
- Which spending categories carry the most budget?
- How does vendor spend trend month over month?
- What is the ratio of paid vs pending vs disputed invoices?

## Tools Used
- Microsoft Excel Online
- Pivot Tables
- SUMIF, COUNTA formulas
- Line Chart visualization

## Files
- `excel-vendor-spend-analysis.xlsx` -- Full workbook with data, pivots, and dashboard

## Key Findings (Mock Data)
- Total spend: $225,850 across 15 invoices
- Only $112,100 paid -- 50% of total spend still outstanding
- Professional Services largest category at $83,000
- Finance department carrying highest outstanding balance at $94,200
